# Table state

# Try
1. Download this project from `github` and change directory.
```shell
git clone https://github.com/AlfonsoG-dev/TableState
cd TableState
```
2. Install package dependencies with `npm`.
```
npm install
```
3. Run project with `npm`.
```
npm run dev
```
